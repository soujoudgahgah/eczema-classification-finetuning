# Eczema Classification using Fine-Tuned Keras Model

This repository contains the fine-tuning code, configurations, and metadata for a Deep Learning model designed to classify eczema from medical images.

## 🚀 Project Overview
- **Task:** Medical Image Classification (Eczema Detection).
- **Framework:** Keras / TensorFlow.
- **Core Work:** Fine-tuning a pre-trained convolutional neural network (CNN) to optimize classification accuracy on skin disease datasets.

## 📁 Repository Structure
- `eczema_model_finetuning.ipynb`: The core Jupyter notebook containing the dataset mounting, network setup, fine-tuning phase, and model fitting.
